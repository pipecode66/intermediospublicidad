# Logos de Intermedios

La web usa por defecto estos SVG:

- `intermedios-blanco.svg` (versión blanca)
- `intermedios-multicolor.svg` (versión naranja + negro)
- `intermedios-negro.svg` (versión negra)

Si luego quieres reemplazarlos por tus PNG oficiales, conserva los mismos nombres base y actualiza las rutas en `index.html`.
